  - 实时保存，立即生效

- ✅ **改进用户体验** - 优化保存流程和交互反馈
  - 更流畅的模态框交互
  - 更清晰的保存状态提示

## v1.0.0 (初始版本)

### 核心功能

- ✅ **基础下载功能**
  - 支持 Windows、macOS、Linux 三个平台
  - 多种存储方式（Link、R2、S3）
  - 下载密码保护
  - 下载次数统计

- ✅ **管理后台**
  - JWT 认证系统
  - 下载项管理（添加、删除、启用/禁用）
  - 完整的管理界面

- ✅ **分类系统**
  - 自定义分类（图标、颜色、描述）
  - 分类关联下载项
  - 分类筛选功能

- ✅ **配置指引系统**
  - 支持 Markdown 格式
  - 智能识别复制/打开动作
  - 弹窗展示详细指引

- ✅ **批量操作**
  - 批量选择/全选
  - 批量移动到分类
  - 批量删除

- ✅ **拖放排序**
  - 分类拖放排序
  - 直观的拖放界面

- ✅ **安全特性**
  - Cloudflare Turnstile 人机验证
  - 失败次数保护
  - 短期下载令牌
  - 文件访问控制

### 技术特性

- ⚡ **边缘计算** - 基于 Cloudflare Workers 的全球分发
- 🎨 **可爱动漫风格 UI** - 柔和的粉紫色渐变配色，丰富的动画效果
- 📦 **多存储支持** - 外部链接、Cloudflare R2、自定义 S3 存储
- 🔐 **多重安全防护** - 密码保护 + Turnstile 人机验证

## 更新规范

### 版本号规则

遵循语义化版本控制（Semantic Versioning）：

- **主版本号（Major）**：不兼容的 API 修改
- **次版本号（Minor）**：向下兼容的功能性新增
- **修订号（Patch）**：向下兼容的问题修正

### 更新类型

- 🆕 **新增功能**：New Feature
- ✨ **功能改进**：Enhancement
- 🐛 **Bug 修复**：Bug Fix
- 📝 **文档更新**：Documentation
- 🔧 **配置调整**：Configuration
- ⚡ **性能优化**：Performance
- 🔒 **安全修复**：Security Fix
